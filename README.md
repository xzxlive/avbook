#### 使用方法:

   >1.解压到 D:\laragon <br>
    2.打开 D:\laragon\laragon.exe ,点击 `启动所有`<br>
    3.点击 `网站` 或者访问 <http://avbook.test>  enjoy it
    
   >ps:<br>
     Ⅰ. 如果没有解压到 D:\laragon 需自行修改Nginx配置，php.ini配置，并在hosts文件新增一行 ：`127.0.0.1      avbook.test` <br>
     Ⅱ. laragon终端启动爬虫前先 `git pull ` 更新到最新版本。
